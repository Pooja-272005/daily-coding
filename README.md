# Day 1 - Repeated Element in Array (Java)

## Description
This program finds the element that is repeated N times in an integer array.

## Approach
- A frequency array is used to count occurrences of each number.
- If any number appears more than once, it is returned as the answer.

## Files
- `Solution.java` : Java program to find the repeated element.

## Example
Input:
1 2 3 3

Output:
3

## Concepts Used
- Arrays
- Frequency counting
- Loops
- Conditional statements

## Language
Java
